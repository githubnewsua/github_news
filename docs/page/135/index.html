Источник: Новое Время https://nv.ua/ukraine/events/ubegayushchie-okkupanty-ukrali-iz-hersonskoy-oblasti-17-skoryh-novosti-ukrainy-50285740.html Об этом заявил заместитель министра здравоохранения Украины Алексей Яременко во время национального телемарафона.
